===================================
Ecuador's Payment
===================================

Instala las configuraciones necesarias para registrar y administrar las formas de pago comunes para Ecuador.

Installation
============

To install this module, you need to:

* Add the module to your addons path.
* Install the module as usual.

Instalación
===========

Para instalar este módulo se debe:

* Agregar el módulo al directorio de addons.
* Instalarlo de manera regular.

Configuration
=============

This module doesn't requiere any configuration.

Configuración
=============

Este módulo no requiere configuraciones.

Problemas conocidos y planificación
===================================

* No se ha configurado la posibilidad de que un anticipo sea devuelto en efectivo, solo a través del pago de facturas.
